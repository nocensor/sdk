# Contributing to @nocensor/sdk

## This is a read-only mirror

This repository (`github.com/nocensor/sdk`) is a **published snapshot** of the SDK package. It is automatically synced from the private development monorepo on each release.

**Pull requests opened here cannot be merged directly.** If you want to contribute a change, please:

1. Open an issue here describing the bug or feature request.
2. We will port confirmed fixes into the monorepo and release them in the next version.

## Reporting bugs

Open an issue on this repo with:

- SDK version (`npm list @nocensor/sdk`)
- Runtime (Node.js version, browser, edge runtime)
- Minimal reproduction — the smaller the better
- Expected vs. actual behavior

## API issues

If you believe the issue is with the API itself (not the SDK client), email support@nocensor.ai.
